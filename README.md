# 날;기록 (Narr;archive)

## "날(Me) 기록하며 쌓이는 나만의 Narrative"


🚀 **프로젝트 소개**

- 날;기록 (Narr;archive)은 소소한 일상부터 창작물, 그리고 날카로운 비평까지,

  나라는 사람을 구성하는 파편들을 체계적으로 모아두고 관리하기 위한 개인 아카이빙 웹 서비스입니다.
  
- 단순한 일기장을 넘어, 기록의 성격에 따라 4가지 카테고리로 나누어 나만의 서사를 기록하는 공간입니다.


✨ **4가지 카테고리**

- 📅 날(Day)의 기록: 달력을 기반으로 소소한 일상과 기억을 매일매일 기록합니다.

- 🕊️ 날(Fly) 기록: 소설, 시, 산문, 가사 등 직접 창작한 결과물을 관리하고 다듬는 나만의 서재입니다.

- ⚔️ 날(Blade)카로운 기록: 영화, 책, 음악 등 감상한 콘텐츠에 대해 솔직한 평점과 비평을 남기는 아카이브입니다.

- ✏️ 날(Raw)것의 기록: 스쳐 지나가는 생각이나 낙서를 포스트잇처럼 가볍게 붙여두는 실시간 생각 저장소입니다.
