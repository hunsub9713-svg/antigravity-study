# 📜날;기록 (Narr;archive)

> 날(Me) 기록하며 쌓이는 나만의 Narrative


## 📢 소개

- '날;기록 (Narr;archive)'은 소소한 일상부터 창작물, 그리고 날카로운 비평까지,

  나라는 사람을 구성하는 파편들을 체계적으로 모아두고 관리하기 위한 개인 아카이빙 웹 서비스입니다.
  
- 단순한 일기장을 넘어, 기록의 성격에 따라 4가지 카테고리로 나누어 나만의 서사를 기록하는 공간입니다.


## ✨ 4가지 카테고리

- 📅 날(Day)의 기록: 달력을 기반으로 소소한 일상과 기억을 매일매일 기록합니다.

- 🕊️ 날(Fly) 기록: 소설, 시, 산문, 가사 등 직접 창작한 결과물을 관리하고 다듬는 나만의 서재입니다.

- ⚔️ 날(Blade)카로운 기록: 영화, 책, 음악 등 감상한 콘텐츠에 대해 솔직한 평점과 비평을 남기는 아카이브입니다.

- ✏️ 날(Raw)것의 기록: 스쳐 지나가는 생각이나 낙서를 포스트잇처럼 가볍게 붙여두는 실시간 생각 저장소입니다.


## 🔓 실행 방법

- 계정 생성 및 로그인 화면에서 아이디와 비밀번호를 입력한 후 계정이 없다면 계정을 생성하고, 계정이 있다면 로그인한다.

- 원하는 기록 카테고리를 클릭하여 기록한다.

## 💡 수정 및 보완할 점

- GitHub Pages는 정적 사이트만 배포되기 때문에 별도의 서버나 데이터베이스를 둘 수 없어, 계정 정보 및 기록 데이터는 브라우저의 localStorage에 저장하는 방식으로 구현함.

  이로 인해 같은 기기·같은 브라우저에서만 데이터가 유지되며, 기기를 변경하거나 브라우저 데이터를 삭제하면 초기화됨.

- 추후 보완한다면 Firebase 등 외부 백엔드 연동을 통해 여러 기기에서도 동일한 계정으로 데이터를 동기화할 수 있도록 개선 가능.

- 과제 제출용이기에 복잡한 보안 절차(비밀번호 암호화 등)는 일부러 구현하지 않음.
